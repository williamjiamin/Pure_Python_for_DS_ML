This chapter shows the core ideas behind XGBoost-style learning without external ML frameworks.

## Core objective
For binary classification with logits `F(x)`:

- Probability: `p = sigmoid(F)`
- Loss per sample: `L = -[y log p + (1-y) log (1-p)]`

At each boosting round, build a tree on Taylor terms:
- Gradient: `g_i = p_i - y_i`
- Hessian: `h_i = p_i(1 - p_i)`

For a split into left/right leaves:
- `G_L = sum(g_i in L)`, `H_L = sum(h_i in L)`
- `G_R = sum(g_i in R)`, `H_R = sum(h_i in R)`
- `G = G_L + G_R`, `H = H_L + H_R`

Split gain:

`Gain = 0.5 * [ G_L^2/(H_L+lambda) + G_R^2/(H_R+lambda) - G^2/(H+lambda) ] - gamma`

Leaf weights:

`w_L = -G_L/(H_L+lambda)`, `w_R = -G_R/(H_R+lambda)`

## Practical tuning notes
- `lambda` controls leaf weight shrinkage.
- `gamma` penalizes weak splits.
- `min_leaf` prevents unstable leaves on tiny partitions.
- `eta` (learning rate) should be small enough to avoid oscillation.
